---

## TOP-Image

## 概要

勉強時間を計測及び管理し、勉強時間や勉強量を可視化できるアプリです。

## URL

##　開発に至った背景

自身がプログラミングを学習する上で「学習を習慣化したい！」「効率よく勉強をしたい！」「勉強を続けたが成果が見えない」などと思っていました。

そのような状況の中で、学習を管理できるアプリケーションを思いつきました。

勉強することを楽しみモチベーションをあげることのできるサービスを作りたいという思いでこのアプリケーションを開発いたしました。
